Welcome to Azure Functions

Azure Functions is a serverless compute service that lets you run event-triggered code without having to explicitly provision or manage infrastructure.

You can use Azure Functions to create PDF files from HTML using PuppeteerSharp, a .NET port of the popular Node.js library Puppeteer.

To learn more about Azure Functions, visit the official documentation.
